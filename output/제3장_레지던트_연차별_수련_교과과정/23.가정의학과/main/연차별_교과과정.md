|:-----|:--------|:------------------------------------------------|
| 1    | 환자취급범위  | 퇴원환자 60명 이상(연간 실인원, 가정의학과 환자 10명 포함)            |

|      |         | - 3년간의 수련 기간동안 가정의학과 외래 환자 및 입원 환자 진료를 통하여      |
|      |         | 지속적인 진료와 가족중심, 환자중심적인 포괄적 진료를 제공하는 경험을          |

|      |         | - 임상에서 흔한 윤리적 문제들에 대한 대처능력을 높이고, 전문직윤리관         |

|      |         | 2. 필수 전공과목                                      |

|      |         | 수 있도록 가정의학 전속수련 및 파견 순환 수련의 시간과 내용이 배정되고        |
|      |         | 계획되어야 함.                                        |

|      |         | 수련.                                             |
|      |         | - 병력청취, 신체검사, 검사실 검사 소견의 청구 및 해석능력 향상           |

|      |         | - 단순 X-선, 심전도, 폐기능 검사 등 각종 기능검사 판독능력 향상         |

|      |         | - 신생아실 근무를 포함하여 입원환자 및 외래환자 진료 경험을 충분히          |

|      |         | - 여성의 임신, 분만, 출산이 가족에 미치는 정서적이고 생물학적인 영향에       |

|      |         | 갖출 수 있도록 가정의학 전속수련 및 파견 순환 수련의 시간과 내용이          |
